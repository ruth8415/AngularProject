1. הוספת חבר לצוות לפי שם משתמש
===========================================

✅ נוסף API חדש להוספת חבר לצוות לפי שם משתמש:
   - Endpoint: POST /api/teams/:teamId/members
   - Body: { "username": "שם_משתמש" }
   - הקוד תומך גם ב-userId וגם ב-username

✅ נוסף UI להוספת חברים לצוות:

   - שדה קלט לשם משתמש
   - הודעות שגיאה והצלחה

===========================================

✅ נוסף API למחיקת פרויקט:

   - מעדכן אוטומטית את רשימת הפרויקטים לאחר מחיקה

קבצים שעודכנו:

3. קבלת פרטי משתמש לפי טוקן
===========================================

✅ נוסף API לקבלת פרטי המשתמש המחובר:
   - Endpoint: GET /api/auth/me
   - Headers: Authorization: Bearer <token>
   - מחזיר: { id, email, name }

✅ עדכון ב-AuthService:
   - נוספה מתודה: getCurrentUser()
   - מעדכן אוטומטית את ה-currentUser signal

קבצים שעודכנו:

===========================================
סיכום Endpoints זמינים
===========================================

Auth:
   - POST /api/auth/register
   - POST /api/auth/login
   - GET /api/auth/me ⭐ חדש

Users:
   - GET /api/users ⭐ חדש (מחזיר כל המשתמשים עם ID + NAME + EMAIL)

Teams:
   - GET /api/teams
   - POST /api/teams
   - POST /api/teams/:teamId/members (תומך ב-username או userId) ⭐ מעודכן
   - GET /api/teams/:teamId/members ⭐ חדש (מחזיר את כל חברי הצוות)
   - DELETE /api/teams/:teamId/members/:userId ⭐ חדש (מוחק חבר מצוות)
   - DELETE /api/teams/:teamId ⭐ חדש (מוחק צוות)

Projects:
   - GET /api/projects
   - POST /api/projects
   - PATCH /api/projects/:projectId ⭐ חדש (מעדכן פרויקט)
   - DELETE /api/projects/:projectId ⭐ חדש (מוחק פרויקט)

Tasks:
   - GET /api/tasks?projectId=:id
   - POST /api/tasks
   - PATCH /api/tasks/:id
   - DELETE /api/tasks/:id

Comments:
   - GET /api/comments?taskId=:id
   - POST /api/comments

Health:
   - GET /api/health

===========================================
הערות למפתחים
===========================================

1. כל ה-API endpoints דורשים authentication מלבד register ו-login
2. יש לשלוח את ה-token ב-header: Authorization: Bearer <token>
3. כל השגיאות מוחזרות בפורמט: { message: "הודעת שגיאה" }
4. כל התאריכים מוחזרים בפורמט ISO 8601
